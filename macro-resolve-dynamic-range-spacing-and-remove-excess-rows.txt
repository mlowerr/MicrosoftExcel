Macro Tip: Resolve #SPILL! Space Constraints and Then Clean Excess Blank Rows

Use these macros when dynamic array formulas in column A return `#SPILL!` because they do not have enough room to expand, and you want a follow-up cleanup pass.

What you get:
- Automatic insertion of rows beneath spill errors until each dynamic array can spill.
- A cleanup pass that removes extra consecutive blank rows.
- A wrapper macro to run both steps together.

Why you would use this:
- Quickly repair space-related spill issues in imported or frequently reshaped sheets.
- Reduce manual row insert/delete work in large datasets.
- Keep worksheet layout cleaner after automated spill fixes.

Sub InsertBlankRowsAndDeleteEmptyCells()
    InsertBlankRowsToResolveSpillErrors
    RemoveBlankRows
End Sub

Sub InsertBlankRowsToResolveSpillErrors()
    Dim lastRow As Long
    Dim i As Long
    Dim blankCount As Integer

    ' Find the last row with data in column A
    lastRow = Cells(Rows.Count, 1).End(xlUp).Row

    ' Loop through each row
    For i = lastRow To 1 Step -1
        If Cells(i, 1).Text = "#SPILL!" Then
          While (Cells(i, 1).Text = "#SPILL!")
            Rows(i + 1).Insert Shift:=xlDown
          Wend
        End If
    Next i
    
    RemoveBlankRows
    
End Sub

Sub RemoveBlankRows()
        Dim lastRow As Long
        
        lastRow = Cells(Rows.Count, 1).End(xlUp).Row

        ' Check for two consecutive blank cells
        blankCount = 0
        
        For i = lastRow To 1 Step -1
            If Cells(i, 1).Text = "" Then
                blankCount = blankCount + 1
            Else
                blankCount = 0
            End If

            ' If two consecutive blank cells are found, delete the row
            If blankCount = 2 Then
                Rows(i).Delete
                RemoveBlankRows
            End If
        Next i

End Sub
